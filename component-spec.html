接入流程

1
下载模板
复制 component-template.html，改成你自己的组件。模板里有完整注释，按提示填写即可。
2
按规范开发
看下面的「必填信息」和「尺寸规范」，画布宽度统一 750px。颜色尽量用 config 接收，不要写死。
3
发给朱彦霖
把 HTML 文件发过来，我来对接接入工具。你不需要懂 React，我会处理所有工程部分。
4
更新迭代
接入后想改效果，直接改你的 HTML 重新发即可。接口不变的话工具侧基本不需要改动。

必填 Meta 标签

标签说明示例值
sg-component 固定填 true，用于识别这是个工具组件 必填 true
sg-name 组件中文名，会显示在工具界面里 必填 头图动效
sg-id 组件英文 ID，小写 + 短横线，全局唯一 必填 header-anim
sg-canvas-width 画布宽度（px），通常固定 750 必填 750
sg-canvas-height 画布高度（px），见下方尺寸表 必填 424
sg-version 你的版本号，改动时递增 选填 1.0.0
sg-author 你的名字 选填 张三 
<meta name="sg-component"    content="true">
<meta name="sg-name"         content="头图动效">
<meta name="sg-id"           content="header-anim">
<meta name="sg-canvas-width"  content="750">
<meta name="sg-canvas-height" content="424">

配置数据结构（config 对象）

💡 工具会通过 SG_SET_CONFIG 消息把这个对象发给你的组件。用到哪些字段就读哪些，不需要全部用。
{
  // ── 主题色（跟随会场全局换色联动）──────────────────────
  theme: {
    primaryColor:  '#fad900',   // 美团黄
    accentColor:   '#ffa203',   // 暖橙
    bgColor:       '#FFD9E5',   // 当前会场背景色
  },

  // ── 文案（工具配置面板的输入框内容）────────────────────
  texts: {
    title:    '节日限定花',   // 主标题
    subtitle: '心意即刻达',   // 副标题
    desc:     '直降低至5折起', // 描述
  },

  // ── 图片（用户上传后的 base64 或 URL）────────────────
  images: {
    main: '',   // 主图
    logo: '',   // Logo
  },

  // ── 你自己定义的参数（自由扩展）─────────────────────
  custom: {
    animSpeed: 1.0,   // 动效速度倍率
    loop:      true,  // 是否循环
  },
}

消息接口（postMessage API）

工具 → 组件（你需要监听的）
SG_SET_CONFIG
更新配置数据，触发重新渲染
SG_EXPORT
请求导出 PNG，需要回传 dataUrl
SG_PREVIEW
请求刷新预览图，需要回传 dataUrl
组件 → 工具（你需要发送的）
SG_READY
页面加载完成后发送，告知工具准备就绪
SG_EXPORT_RESULT
响应导出请求，携带 PNG dataUrl
SG_PREVIEW_RESULT
响应预览请求，携带预览图 dataUrl
// 监听工具发来的消息（必须）
window.addEventListener('message', (event) => {
  const { type, payload } = event.data || {}

  switch (type) {
    case 'SG_SET_CONFIG': render(payload); break
    case 'SG_EXPORT':
      window.parent.postMessage({ type: 'SG_EXPORT_RESULT', dataUrl: exportPNG() }, '*')
      break
  }
})

// 告知工具已就绪（必须，在 load 事件里发）
window.addEventListener('load', () => {
  render(config) // 先渲染一次默认状态
  window.parent.postMessage({ type: 'SG_READY', meta: { name: '头图动效', id: 'header-anim' } }, '*')
})

注意事项

  • 画布宽度必须 750px（手机端展示标准宽度）
  • 导出的 PNG 分辨率建议 @2x（canvas 内部 1500px 再缩回 750px）
  • 颜色、文案用 config 接收，不要写死，方便工具面板调节
  • 尽量不依赖外部 CDN（网络不稳定会加载失败），把资源内联进 HTML
  • 如果用 GSAP，工具里已有 GSAP 3，可以直接用（告知我版本）
  • 不要用 alert / confirm / prompt，会阻塞工具主界面
  • 动效组件记得在 SG_EXPORT 触发时暂停动画，抓静态帧导出
