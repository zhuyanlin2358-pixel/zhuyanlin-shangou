← 查看完整规范 下载模板文件
👋 这是一个 HTML 模板文件，用于开发闪购工作台的外部组件。
使用方法：点右上角「下载模板文件」→ 用代码编辑器打开 → 把 TODO 部分改成你的内容 → 发给朱彦霖接入工具。

模板基本信息

画布尺寸
750 × 424 px（头图标准尺寸，可改）
通信方式
window.postMessage（SG_ 前缀）
导出格式
canvas.toDataURL() → PNG
配置接收
监听 SG_SET_CONFIG 消息，获取颜色/文案/图片

模板代码（完整可复制）

HTML · sg-component-template.html 
<!DOCTYPE html>
<html lang="zh-CN">
<head>
<meta charset="UTF-8">

<!-- 必填 meta（工具读取这些信息）-->
<meta name="sg-component"       content="true">
<meta name="sg-name"            content="组件中文名">      <!-- 如：头图动效 -->
<meta name="sg-id"              content="your-comp-id">  <!-- 英文，如：header-anim -->
<meta name="sg-canvas-width"    content="750">
<meta name="sg-canvas-height"   content="424">
<meta name="sg-version"         content="1.0.0">
<meta name="sg-author"          content="你的名字">

<title>组件名称</title>
<style>
  * { box-sizing: border-box; margin: 0; padding: 0; }
  html, body { width: 100%; height: 100%; overflow: hidden; background: transparent; }

  /* ↓↓↓ 你的样式写在这里 ↓↓↓ */
  #canvas { display: block; }
</style>
</head>
<body>

<canvas id="canvas" width="750" height="424"></canvas>

<script>
// ── 当前配置（工具会通过 SG_SET_CONFIG 消息更新）──────────
let config = {
  theme:  { primaryColor: '#fad900', bgColor: '#FFD9E5' },
  texts:  { title: '主标题', subtitle: '副标题' },
  images: { main: '' },
  custom: { /* 你自己的参数 */ },
}

// ── 渲染函数（核心逻辑写在这里）──────────────────────────
function render(cfg) {
  config = { ...config, ...cfg }
  const canvas = document.getElementById('canvas')
  const ctx = canvas.getContext('2d')

  // ↓↓↓ 你的绘制代码写在这里 ↓↓↓
  ctx.fillStyle = config.theme.bgColor
  ctx.fillRect(0, 0, 750, 424)
  ctx.fillStyle = '#333'
  ctx.font = 'bold 48px PingFang SC, sans-serif'
  ctx.textAlign = 'center'
  ctx.textBaseline = 'middle'
  ctx.fillText(config.texts.title, 375, 212)
}

// ── 导出 PNG（不要改这个函数）─────────────────────────────
function exportPNG() {
  return document.getElementById('canvas').toDataURL('image/png')
}

// ── 监听工具消息（不要改这段）────────────────────────────
window.addEventListener('message', (e) => {
  const { type, payload } = e.data || {}
  if (type === 'SG_SET_CONFIG') render(payload)
  if (type === 'SG_EXPORT')
    window.parent.postMessage({ type: 'SG_EXPORT_RESULT', dataUrl: exportPNG() }, '*')
  if (type === 'SG_PREVIEW')
    window.parent.postMessage({ type: 'SG_PREVIEW_RESULT', dataUrl: exportPNG() }, '*')
})

// ── 通知工具已就绪（不要改这段）──────────────────────────
window.addEventListener('load', () => {
  render(config)
  window.parent.postMessage({
    type: 'SG_READY',
    meta: {
      name:   document.querySelector('meta[name="sg-name"]').content,
      id:     document.querySelector('meta[name="sg-id"]').content,
      width:  750, height: 424,
    },
  }, '*')
})
</script>
</body>
</html>
