关键设计规则（Code Rules）
R-001
Canvas @2x 超采样
所有 Canvas 内部 2× 渲染，drawXxx 绘完后 downsample() 缩回 1× 输出 → 高清切图
R-002
楼层颜色串联规则
isDarkBg=true → 文字白色；isDarkBg=false → 文字跟 slot.titleColor；两侧 icon = slotTintFrom（单色）
R-003
组件预览宽度规则
满宽组件（老虎机/楼层/横滑Tab/商品）无侧边留白；红包有 24px 侧边距（Figma实际设计值）
R-004
字体两遍渲染
preloadFonts() 完成前先用占位色渲染一遍（不白屏），字体加载完后精渲一遍（精确文字）
R-005
子组件必须在模块顶层
不能在父组件函数体内定义子组件，否则每次 re-render 重建组件类型，input 必定失焦
R-006
patchConfig 函数式更新
所有 Context 用 prev => ({...prev,...patch}) 函数式更新，防色轮/滑块拖动时 stale closure 导致颜色卡住
