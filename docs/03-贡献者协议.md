# 🔐 贡献者协议 · 内部预览版

> 本项目目前处于**封闭开发阶段**。  
> 核心架构和代码由项目负责人（朱彦霖）独立维护，以确保产品体验的一致性和迭代速度。

---

## 当前合作方式（三种）

### 1. 需求反馈
通过 [需求漏斗](./02-需求漏斗.md) 按格式提交。  
不接受口头需求，不接受无背景的建议。

### 2. 模板贡献
如果你在工具中搭建了一个效果好的活动页面，可以将其沉淀为模板：

1. 在工具中完成页面配置
2. 使用「保存为模板」功能导出 `.template.json` 文件
3. 将文件发给朱彦霖审核
4. 审核通过后加入官方模板库，署名贡献者

**模板审核标准：**
- 符合美团闪购视觉规范
- 有明确的适用场景（如"大促主场 - 热烈红"）
- 在真实活动中验证使用过

### 3. 技术讨论
欢迎对技术方案提出想法（Canvas 渲染、导出逻辑、组件结构等），  
但**最终决策权在项目负责人**，以保证代码库的一致性。

---

## 为什么暂时封闭？

- **快速迭代需要**：早期产品需要频繁调整架构，多人同时修改会产生冲突和债务
- **规范尚在建立**：技术栈（React 18 + TypeScript + Vite）和组件规范仍在完善中
- **首要目标**：让工具稳定跑起来，解决真实业务问题，而不是建立完美的协作流程

> **先把工具做"对"，再把它做"大"。**
